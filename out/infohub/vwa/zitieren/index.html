Zitieren & Literaturverwaltung — VWA / ABA

Korrektes Zitieren ist eine der wichtigsten Anforderungen an eine wissenschaftliche Arbeit. Auf dieser Seite erfahren Sie, wie Sie Quellen richtig angeben, KI-Nutzung dokumentieren und mit Zotero Ihre Literatur effizient verwalten.

Richtig Zitieren

Der Zitierstil wird in Absprache mit der Betreuungsperson gewählt und muss durchgehend einheitlich angewendet werden.

StilTypisch fürBeispiel
APA (In-Text)Naturwissenschaften, Psychologie, IT, Sport(Müller, 2020, S. 15)
Fußnoten (dt. Stil)Geisteswissenschaften, Geschichte, PhilosophieVgl. Müller, Maria: Titel. Wien 2020, S. 15.

KI-Nutzung (ChatGPT & Co.)

  • Generative KI darf als Hilfsmittel verwendet werden
  • Alle KI-Nutzung muss dokumentiert und zitiert werden
  • Vollständige Prompts und Ausgaben müssen in den Anhang
  • Eine unterschriebene KI-Nutzungserklärung ist Pflicht
  • Beispiel-Zitation: (OpenAI ChatGPT, "Titel des Chats", 15.01.2026, Anhang A)
⚠️KI ersetzt nicht das eigene Denken
KI-generierte Texte dürfen nicht unverändert übernommen werden. Prüfen Sie jede KI-Ausgabe auf Korrektheit — generative Modelle können plausibel klingende, aber falsche Informationen erzeugen ("Halluzinationen"). Die eigenständige wissenschaftliche Leistung muss klar erkennbar bleiben.

Zotero — Kostenlose Literaturverwaltung

Was ist Zotero?

Zotero ist eine kostenlose Open-Source-Literaturverwaltung. Sie speichert Quellen, organisiert sie in Sammlungen und generiert automatisch Zitate und Literaturverzeichnisse in Word und LaTeX. Empfohlen von der Schule für alle VWA-Arbeiten.

Installation

1

Zotero herunterladen

Download von zotero.org — verfügbar für Windows, macOS, Linux. Kostenlos, keine Registrierung nötig.
2

Browser Connector installieren

Zotero Connector für Chrome, Firefox oder Edge installieren. Ermöglicht das Speichern von Quellen direkt aus dem Browser mit einem Klick.
3

Konto erstellen (optional)

Ein kostenloses Konto auf zotero.org synchronisiert die Bibliothek zwischen Geräten (300 MB kostenlos, für VWA meist ausreichend).

Quellen sammeln

MethodeSo geht's
Browser-ButtonAuf einer Webseite/Buch/Artikel den Zotero Connector-Button klicken — Quelle wird automatisch mit allen Metadaten gespeichert
ISBN/DOIIn Zotero: Zauberstab-Icon — ISBN oder DOI eingeben — Quelle wird automatisch gefunden
ManuellIn Zotero: Grünes + — Quellentyp wählen (Buch, Zeitschrift, etc.) — Felder ausfüllen
PDF importierenPDF in Zotero ziehen — Metadaten werden automatisch erkannt
ℹ️Tipp
Speichern Sie jede Quelle sofort in Zotero, wenn Sie sie finden. Quellen später wiederzufinden kostet unnötig Zeit.

Organisieren

  • Sammlungen: Erstellen Sie Ordner für Kapitel (z.B. "Theorie", "Methodik", "Ergebnisse")
  • Tags: Vergeben Sie Schlagwörter (z.B. "Klimawandel", "Statistik", "Interview")
  • Notizen: Fügen Sie zu jeder Quelle Ihre persönlichen Anmerkungen hinzu — das erleichtert das spätere Schreiben

Zotero + Microsoft Word

1

Plugin prüfen

Das Zotero Word Plugin wird automatisch bei der Zotero-Installation mitinstalliert. In Word erscheint ein neuer Tab "Zotero". Falls nicht: Zotero → Einstellungen → Zitieren → Textverarbeitung → "Microsoft Word Plugin neu installieren".
2

Zitierstil wählen

Beim ersten Zitat fragt Zotero nach dem Stil. Für VWA empfohlen: "APA 7th Edition" (naturwiss.) oder "Chicago Manual of Style 17th (note)" (geisteswiss.). Der Stil kann jederzeit nachträglich geändert werden.
3

Zitat einfügen

Cursor an die gewünschte Stelle → Zotero Tab → "Add Citation" → Quelle suchen und auswählen → Seitenzahl eingeben → Enter. Das Zitat wird automatisch formatiert.
4

Literaturverzeichnis erstellen

Cursor ans Ende der Arbeit → Zotero Tab → "Add Bibliography" → Das Verzeichnis wird automatisch generiert und aktualisiert sich bei neuen Zitaten.
5

Vor Abgabe: Refresh

Zotero Tab → "Refresh" klicken, um sicherzustellen dass alle Zitate und das Verzeichnis aktuell sind.
⚠️Zitate nicht manuell ändern
Ändern Sie Zitate nicht manuell im Text! Verwenden Sie immer den Zotero-Tab in Word. Manuelle Änderungen werden beim nächsten Refresh überschrieben.

Zotero + LaTeX (Better BibTeX)

1

Better BibTeX installieren

Download von retorque.re/zotero-better-bibtex. In Zotero: Werkzeuge → Add-ons → Zahnrad → "Install Add-on From File" → .xpi-Datei auswählen → Zotero neu starten.
2

Citation Keys konfigurieren

Better BibTeX → Einstellungen → Citation Key Format: [auth][year] generiert Keys wie Mueller2023. Diese Keys verwenden Sie in LaTeX als \cite{Mueller2023}.
3

Automatischen .bib-Export einrichten

Rechtsklick auf Ihre VWA-Sammlung → "Export Collection" → Format: "Better BibLaTeX" → Haken bei "Keep updated" → Speichern als literatur.bib im LaTeX-Projektordner. Die Datei wird automatisch aktualisiert wenn Sie Quellen hinzufügen.
4

In LaTeX verwenden

In der Präambel: \addbibresource{literatur.bib}. Im Text: \cite{Mueller2023} oder \parencite{Mueller2023}. Am Ende: \printbibliography. Die BGZ-Vorlage hat das bereits vorkonfiguriert.
✅Automatischer Export
Mit Better BibTeX + automatischem Export müssen Sie die .bib-Datei nie manuell bearbeiten. Neue Quellen in Zotero erscheinen automatisch in LaTeX.

Quellenqualität

Goldene Regel: Je näher eine Quelle am Original ist, desto besser.

BevorzugenVermeiden/Vorsicht
Peer-reviewed FachzeitschriftenWikipedia (nur als Einstieg)
Offizielle Dokumente, GesetzeBlog-Beiträge
Lehrbücher bekannter AutorenYouTube-Videos
OriginalstudienKI-generierte Texte (Halluzinationsgefahr)
Statistiken (Statistik Austria, Eurostat)Unbelegte Behauptungen

Richtwerte Quellenzahl

AbschnittRichtwert
Einleitung3–5 Quellen
Theoretischer Teil10–20 Quellen
Empirischer Teil3–8 Quellen (Methodik, Vergleichsstudien)
FazitVerweise auf bereits zitierte Quellen
Gesamtca. 15–30 Quellen
ℹ️Qualität vor Quantität
15 relevante Quellen sind besser als 50 themenfremde. Achten Sie darauf, dass Ihre Quellen aktuell, wissenschaftlich und für Ihre Forschungsfrage relevant sind.
